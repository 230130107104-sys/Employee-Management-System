# 👨‍💼 Employee Management System

A simple Java console-based application that demonstrates Object-Oriented Programming (OOP) concepts and CRUD (Create, Read, Update, Delete) operations. This project allows users to manage employee records efficiently through a menu-driven interface.

---

## 📌 Project Overview

The Employee Management System is a Core Java application developed to manage employee information using an interactive console interface. It helps users perform various operations such as adding, viewing, updating, searching, and deleting employee records.

This project is designed to strengthen Java fundamentals, Object-Oriented Programming concepts, Collections Framework, and user interaction through console-based applications. :contentReference[oaicite:0]{index=0}

---

## 🚀 Features

- ➕ Add a new employee
- 📋 View all employees
- 🔍 Search employee by ID
- ✏️ Update employee details
- ❌ Delete employee record
- 📊 Display total number of employees
- 🚪 Exit the application
